🎨 모든 패키지 테스트

🌐 Web Icons 패키지

1. 전역 함수 테스트

const icon = icon('Gavel', 'regular', 24);

2. CSS 클래스 테스트

const cssClass = RefineUIIcons.getIconClass('Gavel', 'regular', 24);

3. 직접 함수 테스트

const gavel = Gavel(24);

⚛️ React Icons 패키지

1. React 컴포넌트 테스트

import { Icon } from '@refineui/react-icons';
React 컴포넌트는 브라우저에서 직접 테스트 불가
✅ React 환경에서 사용 가능

2. 유틸리티 함수 테스트

const iconChar = getIconChar('Gavel', 'regular', 24);

📱 React Native Icons 패키지

1. React Native 컴포넌트 테스트

import { Icon } from '@refineui/react-native-icons';
React Native 컴포넌트는 브라우저에서 직접 테스트 불가
✅ React Native 환경에서 사용 가능

2. 유틸리티 함수 테스트

const iconChar = getIconChar('Gavel', 'regular', 24);

🌍 Icon CDN 패키지

1. CDN URL 테스트

const cssUrl = getCSSUrl('regular');

2. HTML 생성 테스트

const html = getIconHTML('Gavel', 'regular', 24);

3. 직접 함수 테스트

const gavel = Gavel(24);
