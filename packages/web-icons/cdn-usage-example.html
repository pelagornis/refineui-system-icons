🎨 RefineUI System Icons - CDN 사용 예시

1. CDN CSS 로드

HTML head 태그에 다음을 추가하세요:

<!-- Regular 스타일 아이콘 CSS --> <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/@refineui/system-icons@latest/dist/refineui-system-icons-regular.css"> <!-- Filled 스타일 아이콘 CSS --> <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/@refineui/system-icons@latest/dist/refineui-system-icons-filled.css">

2. CDN JavaScript 로드

HTML body 태그 끝에 다음을 추가하세요:

<script src="https://cdn.jsdelivr.net/npm/@refineui/system-icons@latest/dist/index.umd.js"></script>

3. 아이콘 사용 예시

CSS 클래스 방식

<span class="ic_refineui_gavel_24_regular"></span> <span class="ic_refineui_gavel_24_filled"></span>

JavaScript 함수 방식

// 아이콘 HTML 생성 const gavelIcon = getIconHTML('Gavel', 'regular', 24, 'red'); const gavelFilledIcon = getIconHTML('Gavel', 'filled', 32, 'blue'); // 직접 함수 사용 const gavel = Gavel(24, 'green'); const gavelFilled = GavelFilled(32, 'purple');

유틸리티 함수

// 아이콘 문자 가져오기 const iconChar = getIconChar('Gavel', 'regular', 24); const fontFamily = getFontFamily('regular'); // CSS 클래스 가져오기 const cssClass = getIconClass('Gavel', 'regular', 24);

4. 사용 가능한 함수들

// CDN URL 생성 getCSSUrl('regular') // CSS 파일 URL getFontUrl('regular') // 폰트 파일 URL getCSSLinkTag('regular') // HTML link 태그 // 아이콘 생성 getIconHTML('Gavel', 'regular', 24, 'red') getIconClassHTML('Gavel', 'regular', 24) // 유틸리티 getIconChar('Gavel', 'regular', 24) getIconClass('Gavel', 'regular', 24) getFontFamily('regular') // 직접 함수 (434개 아이콘 모두 사용 가능) Gavel(24, 'red') GavelFilled(32, 'blue') Add(20, 'green') Home(28, 'purple') // ... 등등

5. 실제 사용 예시

<!DOCTYPE html> <html> <head> <title>RefineUI Icons Demo</title> <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/@refineui/system-icons@latest/dist/refineui-system-icons-regular.css"> <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/@refineui/system-icons@latest/dist/refineui-system-icons-filled.css"> </head> <body> <h1>아이콘 예시</h1> <!-- CSS 클래스 방식 --> <span class="ic_refineui_gavel_24_regular"></span> <span class="ic_refineui_gavel_24_filled"></span> <script src="https://cdn.jsdelivr.net/npm/@refineui/system-icons@latest/dist/index.umd.js"></script> <script> // JavaScript 방식 document.body.innerHTML += getIconHTML('Gavel', 'regular', 24, 'red'); document.body.innerHTML += GavelFilled(32, 'blue'); </script> </body> </html>
