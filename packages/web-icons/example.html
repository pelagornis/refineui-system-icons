🎨 RefineUI System Icons - Flutter Style

📱 Flutter 스타일 아이콘 컴포넌트

Flutter의 Icon 위젯과 유사한 API를 제공하는 React 컴포넌트입니다.

Flutter (Dart)
Icon( Icons.gavel, size: 24, color: Colors.red, )
React (이 패키지)
<FlutterIcon name="Gavel" size={24} color="red" />

🔧 사용법

1. 컴포넌트 방식

import { FlutterIcon, Icon } from '@refineui/web-icons'; // 기본 사용법 <FlutterIcon name="Gavel" size={24} color="red" /> // filled 스타일 <Icon name="Add" style="filled" size={32} color="blue" /> // 클릭 이벤트 <FlutterIcon name="Settings" size={20} color="green" onClick={() => console.log('clicked')} />

2. 함수 호출 방식

import { Gavel, AddFilled, Home } from '@refineui/web-icons'; {Gavel({ size: 32, color: 'blue' })} {AddFilled({ size: 48, color: 'green' })} {Home({ size: 24, color: 'red' })}

3. 유틸리티 함수

import { createIcon } from '@refineui/web-icons'; {createIcon('Gavel', 'filled', { size: 48, color: 'green' })} {createIcon('Add', 'regular', { size: 24, color: 'blue' })}

📦 설치 및 설정

1. 패키지 설치

npm install @refineui/web-icons

2. Font 파일 포함

// CSS 파일 import import '@refineui/web-icons/dist/fonts.css'; // 또는 CDN 사용 <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/@refineui/web-icons/dist/fonts.css">

✨ 특징

  • Flutter 스타일 API: Flutter의 Icon 위젯과 유사한 직관적인 API
  • Font 기반: 벡터 아이콘으로 선명하고 확장 가능
  • 434개 아이콘: 풍부한 아이콘 라이브러리
  • 2가지 스타일: Regular와 Filled 스타일 지원
  • 6가지 크기: 16, 20, 24, 28, 32, 48px 지원
  • TypeScript 지원: 완전한 타입 안전성
  • Tree-shaking: 사용하지 않는 아이콘은 번들에서 제외
