# Flutter 가이드

Flutter 앱에서 RefineUI System Icons를 사용하는 방법을 안내합니다.

## 설치

### pubspec.yaml에 추가

```yaml
dependencies:
  refineui_system_icons: ^0.1.22
```

### 패키지 설치

```bash
flutter pub get
```

## 기본 사용법

### Import

```dart
import 'package:refineui_system_icons/refineui_system_icons.dart';
```

### Icon 위젯 사용

```dart
// Filled 스타일 아이콘
Icon(RefineUIIcons.add_24_filled)

// Regular 스타일 아이콘
Icon(RefineUIIcons.add_24_regular)

// 크기와 색상 지정
Icon(
  RefineUIIcons.add_24_filled,
  size: 32,
  color: Colors.blue,
)
```

## 다양한 크기 사용

```dart
// 16px 아이콘
Icon(RefineUIIcons.add_16_filled)

// 20px 아이콘
Icon(RefineUIIcons.add_20_filled)

// 24px 아이콘
Icon(RefineUIIcons.add_24_filled)

// 28px 아이콘
Icon(RefineUIIcons.add_28_filled)

// 32px 아이콘
Icon(RefineUIIcons.add_32_filled)

// 48px 아이콘
Icon(RefineUIIcons.add_48_filled)
```

## 실제 사용 예제

### AppBar에서 사용

```dart
AppBar(
  title: Text('My App'),
  actions: [
    IconButton(
      icon: Icon(RefineUIIcons.add_24_filled),
      onPressed: () {
        // 추가 버튼 액션
      },
    ),
    IconButton(
      icon: Icon(RefineUIIcons.settings_24_regular),
      onPressed: () {
        // 설정 버튼 액션
      },
    ),
  ],
)
```

### FloatingActionButton에서 사용

```dart
FloatingActionButton(
  onPressed: () {
    // FAB 액션
  },
  child: Icon(RefineUIIcons.add_24_filled),
)
```

### ListTile에서 사용

```dart
ListTile(
  leading: Icon(RefineUIIcons.home_24_filled),
  title: Text('Home'),
  trailing: Icon(RefineUIIcons.chevron_right_24_regular),
  onTap: () {
    // 리스트 아이템 탭 액션
  },
)
```

## 테마와 함께 사용

```dart
Theme(
  data: Theme.of(context).copyWith(
    iconTheme: IconThemeData(
      color: Colors.blue,
      size: 24,
    ),
  ),
  child: Icon(RefineUIIcons.add_24_filled),
)
```

## 성능 최적화 팁

1. **필요한 아이콘만 import**: 전체 라이브러리를 import하지 말고 필요한 아이콘만 사용하세요.

2. **const 사용**: 아이콘을 const로 선언하여 성능을 향상시키세요.

```dart
const Icon(RefineUIIcons.add_24_filled)
```

3. **캐싱**: 자주 사용하는 아이콘은 변수로 캐싱하세요.

```dart
class MyWidget extends StatelessWidget {
  static const IconData addIcon = RefineUIIcons.add_24_filled;
  
  @override
  Widget build(BuildContext context) {
    return Icon(addIcon);
  }
}
```

## 문제 해결

### 아이콘이 표시되지 않는 경우

1. **폰트 파일 확인**: `pubspec.yaml`에 폰트가 올바르게 등록되어 있는지 확인하세요.

2. **패키지 버전 확인**: 최신 버전을 사용하고 있는지 확인하세요.

3. **Hot Reload**: `flutter clean` 후 `flutter pub get`을 실행하세요.

### 빌드 오류가 발생하는 경우

1. **의존성 충돌**: 다른 아이콘 라이브러리와 충돌이 있는지 확인하세요.

2. **Flutter 버전**: Flutter 버전이 호환되는지 확인하세요.
