# Flutter Guide

Learn how to use RefineUI System Icons in Flutter applications.

## Installation

### Add to pubspec.yaml

```yaml
dependencies:
  refineui_system_icons: ^0.1.22
```

### Install Package

```bash
flutter pub get
```

## Basic Usage

### Import

```dart
import 'package:refineui_system_icons/refineui_system_icons.dart';
```

### Using Icon Widget

```dart
// Filled style icon
Icon(RefineUIIcons.add_24_filled)

// Regular style icon
Icon(RefineUIIcons.add_24_regular)

// With size and color
Icon(
  RefineUIIcons.add_24_filled,
  size: 32,
  color: Colors.blue,
)
```

## Different Sizes

## Real-world Examples

### Using in AppBar

```dart
AppBar(
  title: Text('My App'),
  actions: [
    IconButton(
      icon: Icon(RefineUIIcons.add_24_filled),
      onPressed: () {
        // Add button action
      },
    ),
    IconButton(
      icon: Icon(RefineUIIcons.settings_24_regular),
      onPressed: () {
        // Settings button action
      },
    ),
  ],
)
```

### Using in FloatingActionButton

```dart
FloatingActionButton(
  onPressed: () {
    // FAB action
  },
  child: Icon(RefineUIIcons.add_24_filled),
)
```

### Using in ListTile

```dart
ListTile(
  leading: Icon(RefineUIIcons.home_24_filled),
  title: Text('Home'),
  trailing: Icon(RefineUIIcons.chevron_right_24_regular),
  onTap: () {
    // List item tap action
  },
)
```

## Using with Theme

```dart
Theme(
  data: Theme.of(context).copyWith(
    iconTheme: IconThemeData(
      color: Colors.blue,
      size: 24,
    ),
  ),
  child: Icon(RefineUIIcons.add_24_filled),
)
```

## Performance Tips

1. **Import only needed icons**: Don't import the entire library, use only the icons you need.

2. **Use const**: Declare icons as const to improve performance.

```dart
const Icon(RefineUIIcons.add_24_filled)
```

3. **Cache frequently used icons**: Cache frequently used icons in variables.

```dart
class MyWidget extends StatelessWidget {
  static const IconData addIcon = RefineUIIcons.add_24_filled;
  
  @override
  Widget build(BuildContext context) {
    return Icon(addIcon);
  }
}
```

## Troubleshooting

### Icons not displaying

1. **Check font files**: Make sure fonts are properly registered in `pubspec.yaml`.

2. **Check package version**: Make sure you're using the latest version.

3. **Hot Reload**: Run `flutter clean` then `flutter pub get`.

### Build errors

1. **Dependency conflicts**: Check for conflicts with other icon libraries.

2. **Flutter version**: Make sure your Flutter version is compatible.
