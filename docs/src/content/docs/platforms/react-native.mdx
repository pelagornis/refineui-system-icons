# React Native 가이드

React Native 모바일 앱에서 RefineUI System Icons를 사용하는 방법을 안내합니다.

## 설치

### npm으로 설치

```bash
npm install @refineui/react-native-icons
```

### iOS 추가 설정

iOS에서 폰트를 사용하려면 `ios/Runner/Info.plist`에 폰트를 등록해야 합니다:

```xml
<key>UIAppFonts</key>
<array>
    <string>refineui-system-icons-regular.ttf</string>
    <string>refineui-system-icons-filled.ttf</string>
</array>
```

## 기본 사용법

### Import

```jsx
import { Add24Filled, Add24Regular } from '@refineui/react-native-icons';
```

### 컴포넌트 사용

```jsx
import { View, Text } from 'react-native';

function MyComponent() {
  return (
    <View>
      <Add24Filled />
      <Add24Regular />
    </View>
  );
}
```

## 실제 사용 예제

### TouchableOpacity와 함께 사용

```jsx
import { TouchableOpacity, View, Text } from 'react-native';
import { Add24Filled, Settings24Regular } from '@refineui/react-native-icons';

function IconButton({ icon, label, onPress }) {
  return (
    <TouchableOpacity onPress={onPress} style={styles.iconButton}>
      {icon}
      <Text style={styles.buttonText}>{label}</Text>
    </TouchableOpacity>
  );
}

function MyComponent() {
  return (
    <View style={styles.container}>
      <IconButton 
        icon={<Add24Filled />} 
        label="추가" 
        onPress={() => console.log('추가 클릭')} 
      />
      <IconButton 
        icon={<Settings24Regular />} 
        label="설정" 
        onPress={() => console.log('설정 클릭')} 
      />
    </View>
  );
}

const styles = StyleSheet.create({
  container: {
    flex: 1,
    justifyContent: 'center',
    alignItems: 'center',
    gap: 16,
  },
  iconButton: {
    flexDirection: 'row',
    alignItems: 'center',
    gap: 8,
    padding: 12,
    backgroundColor: '#0078d4',
    borderRadius: 8,
  },
  buttonText: {
    color: 'white',
    fontSize: 16,
  },
});
```
