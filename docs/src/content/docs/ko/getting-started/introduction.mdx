# RefineUI System Icons 소개

RefineUI System Icons는 현대적인 애플리케이션을 위한 크로스 플랫폼 아이콘 라이브러리입니다.

## 무엇인가요?

RefineUI System Icons는 다음과 같은 특징을 가진 아이콘 라이브러리입니다:

- **크로스 플랫폼**: Flutter, React, React Native, iOS, Android, Web 지원
- **일관된 디자인**: 모든 플랫폼에서 동일한 시각적 경험 제공
- **다양한 크기**: 16px부터 48px까지 다양한 크기 지원
- **두 가지 스타일**: Regular과 Filled 버전 제공

## 지원 플랫폼

| 플랫폼 | 상태 | 패키지 |
|--------|------|--------|
| Flutter | ✅ | [pub.dev](https://pub.dev/packages/refineui_system_icons) |
| React | ✅ | [npm](https://www.npmjs.com/package/@refineui/react-icons) |
| React Native | ✅ | [npm](https://www.npmjs.com/package/@refineui/react-native-icons) |
| iOS | ✅ | [Swift Package Manager](https://github.com/pelagornis/refineui-system-icons) |
| Android | ✅ | [Maven](https://github.com/pelagornis/refineui-system-icons) |
| Web | ✅ | [CDN](https://github.com/pelagornis/refineui-system-icons) |
