# React 가이드

React 웹 애플리케이션에서 RefineUI System Icons를 사용하는 방법을 안내합니다.

## 설치

### npm으로 설치

```bash
npm install @refineui/react-icons
```

### yarn으로 설치

```bash
yarn add @refineui/react-icons
```

### pnpm으로 설치

```bash
pnpm add @refineui/react-icons
```

## 기본 사용법

### Import

```jsx
import { Add24Filled, Add24Regular } from '@refineui/react-icons';
```

### 컴포넌트 사용

```jsx
// Filled 스타일 아이콘
<Add24Filled />

// Regular 스타일 아이콘
<Add24Regular />

// 크기와 색상 지정
<Add24Filled 
  style={{ 
    width: 32, 
    height: 32, 
    color: '#0078d4' 
  }} 
/>
```

## 다양한 크기 사용

```jsx
import { 
  Add16Filled, 
  Add20Filled, 
  Add24Filled, 
  Add28Filled, 
  Add32Filled,
```

## 실제 사용 예제

### 버튼에서 사용

```jsx
import { Add24Filled, Settings24Regular } from '@refineui/react-icons';

function MyComponent() {
  return (
    <div>
      <button className="icon-button">
        <Add24Filled />
        <span>추가</span>
      </button>
      
      <button className="icon-button">
        <Settings24Regular />
        <span>설정</span>
      </button>
    </div>
  );
}
```

### 네비게이션에서 사용

```jsx
import { 
  Home24Filled, 
  Search24Regular, 
  User24Regular, 
  Settings24Regular 
} from '@refineui/react-icons';

function Navigation() {
  return (
    <nav className="navigation">
      <a href="/home" className="nav-item">
        <Home24Filled />
        <span>홈</span>
      </a>
      <a href="/search" className="nav-item">
        <Search24Regular />
        <span>검색</span>
      </a>
      <a href="/profile" className="nav-item">
        <User24Regular />
        <span>프로필</span>
      </a>
      <a href="/settings" className="nav-item">
        <Settings24Regular />
        <span>설정</span>
      </a>
    </nav>
  );
}
```

### 리스트에서 사용

```jsx
import { 
  ChevronRight24Regular, 
  Home24Filled, 
  Search24Regular 
} from '@refineui/react-icons';

function ListItem({ icon, title, onClick }) {
  return (
    <div className="list-item" onClick={onClick}>
      <div className="list-item-icon">
        {icon}
      </div>
      <div className="list-item-content">
        <span>{title}</span>
      </div>
      <div className="list-item-arrow">
        <ChevronRight24Regular />
      </div>
    </div>
  );
}

function MyList() {
  return (
    <div className="list">
      <ListItem 
        icon={<Home24Filled />} 
        title="홈" 
        onClick={() => console.log('홈 클릭')} 
      />
      <ListItem 
        icon={<Search24Regular />} 
        title="검색" 
        onClick={() => console.log('검색 클릭')} 
      />
    </div>
  );
}
```

## TypeScript 사용

```tsx
import { Add24Filled, Add24Regular } from '@refineui/react-icons';

interface IconButtonProps {
  icon: React.ComponentType<{ style?: React.CSSProperties }>;
  label: string;
  onClick: () => void;
}

function IconButton({ icon: Icon, label, onClick }: IconButtonProps) {
  return (
    <button onClick={onClick} className="icon-button">
      <Icon style={{ width: 24, height: 24 }} />
      <span>{label}</span>
    </button>
  );
}

// 사용 예제
function MyComponent() {
  return (
    <div>
      <IconButton 
        icon={Add24Filled} 
        label="추가" 
        onClick={() => console.log('추가 클릭')} 
      />
      <IconButton 
        icon={Add24Regular} 
        label="추가 (Regular)" 
        onClick={() => console.log('추가 클릭')} 
      />
    </div>
  );
}
```

## 성능 최적화 팁

1. **Tree Shaking**: 필요한 아이콘만 import하여 번들 크기를 줄이세요.

```jsx
// 좋은 예: 필요한 아이콘만 import
import { Add24Filled } from '@refineui/react-icons';

// 피해야 할 예: 전체 라이브러리 import
import * as Icons from '@refineui/react-icons';
```

2. **Memoization**: 자주 사용하는 아이콘을 memoize하세요.

```jsx
import { memo } from 'react';
import { Add24Filled } from '@refineui/react-icons';

const MemoizedIcon = memo(Add24Filled);
```

3. **CSS-in-JS 최적화**: styled-components나 emotion을 사용할 때는 아이콘을 미리 스타일링하세요.

```jsx
import styled from 'styled-components';
import { Add24Filled } from '@refineui/react-icons';

const StyledIcon = styled(Add24Filled)`
  width: 24px;
  height: 24px;
  color: #0078d4;
`;
```

## 문제 해결

### 아이콘이 표시되지 않는 경우

1. **Import 확인**: 아이콘 이름이 정확한지 확인하세요.
2. **CSS 로딩**: 필요한 CSS 파일이 로드되었는지 확인하세요.
3. **번들링**: 빌드 도구가 아이콘을 올바르게 번들링했는지 확인하세요.

### TypeScript 오류가 발생하는 경우

1. **타입 정의**: `@types/react`가 설치되어 있는지 확인하세요.
2. **버전 호환성**: React 버전과 아이콘 라이브러리 버전이 호환되는지 확인하세요.
