# Web 가이드

웹 애플리케이션에서 RefineUI System Icons를 사용하는 방법을 안내합니다.

## 설치

### CDN 사용

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/@refineui/web-icons@0.1.22/dist/refineui-system-icons.css">
```

### npm으로 설치

```bash
npm install @refineui/web-icons
```

## 기본 사용법

### CSS 클래스 사용

```html
<!-- Filled 스타일 아이콘 -->
<i class="refineui-icon refineui-icon-add-24-filled"></i>

<!-- Regular 스타일 아이콘 -->
<i class="refineui-icon refineui-icon-add-24-regular"></i>
```

### 다양한 크기

```html
<!-- 16px 아이콘 -->
<i class="refineui-icon refineui-icon-add-16-filled"></i>

<!-- 20px 아이콘 -->
<i class="refineui-icon refineui-icon-add-20-filled"></i>

<!-- 24px 아이콘 -->
<i class="refineui-icon refineui-icon-add-24-filled"></i>

<!-- 28px 아이콘 -->
<i class="refineui-icon refineui-icon-add-28-filled"></i>

<!-- 32px 아이콘 -->
<i class="refineui-icon refineui-icon-add-32-filled"></i>

<!-- 48px 아이콘 -->
<i class="refineui-icon refineui-icon-add-48-filled"></i>
```

## 실제 사용 예제

### 버튼에서 사용

```html
<button class="icon-button">
  <i class="refineui-icon refineui-icon-add-24-filled"></i>
  <span>추가</span>
</button>

<button class="icon-button">
  <i class="refineui-icon refineui-icon-settings-24-regular"></i>
  <span>설정</span>
</button>
```

### CSS 스타일링

```css
.icon-button {
  display: flex;
  align-items: center;
  gap: 8px;
  padding: 8px 16px;
  border: none;
  background: #0078d4;
  color: white;
  border-radius: 4px;
  cursor: pointer;
}

.icon-button:hover {
  background: #106ebe;
}

.refineui-icon {
  font-size: 24px;
}
```

## JavaScript와 함께 사용

```javascript
// 아이콘 동적 변경
function changeIcon(element, iconName) {
  element.className = `refineui-icon refineui-icon-${iconName}`;
}

// 사용 예제
const button = document.querySelector('.icon-button i');
changeIcon(button, 'add-24-filled');
```
