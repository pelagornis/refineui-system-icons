# 아이콘 검색

RefineUI System Icons에서 원하는 아이콘을 쉽게 찾을 수 있습니다. 5,000+ 아이콘을 이름, 카테고리, 크기, 스타일로 필터링하여 검색할 수 있습니다.

<IconSearch />

## 검색 방법

### 이름으로 검색
아이콘의 이름을 입력하여 검색할 수 있습니다.

예시:
- `add` - 추가 관련 아이콘
- `home` - 홈 관련 아이콘
- `settings` - 설정 관련 아이콘

### 카테고리별 검색
아이콘을 카테고리별로 분류하여 찾을 수 있습니다.

- **기본**: add, remove, edit, delete, save, cancel, confirm
- **네비게이션**: home, back, forward, menu, close, expand, chevron
- **액션**: play, pause, stop, refresh, reload, download, upload, share, copy, paste
- **상태**: check, error, warning, info, success, failure, loading, spinner
- **사용자 인터페이스**: settings, preferences, user, profile, account, notification, bell, help, question

### 크기별 검색
각 아이콘은 다음 크기로 제공됩니다:
- 16px, 20px, 24px, 28px, 32px, 48px

### 스타일별 검색
각 아이콘은 두 가지 스타일로 제공됩니다:
- **Regular**: 기본 스타일
- **Filled**: 채워진 스타일

## 사용법

### Flutter
```dart
import 'package:refineui_system_icons/refineui_system_icons.dart';

Icon(RefineUIIcons.add_24_filled)
Icon(RefineUIIcons.add_24_regular)
```

### React
```jsx
import { Add24Filled, Add24Regular } from '@refineui/react-icons';

<Add24Filled />
<Add24Regular />
```

### Web
```html
<i class="refineui-icon refineui-icon-add-24-filled"></i>
<i class="refineui-icon refineui-icon-add-24-regular"></i>
```
