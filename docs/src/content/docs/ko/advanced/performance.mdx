# 성능 최적화

RefineUI System Icons의 성능을 최적화하는 방법을 안내합니다.

## 번들 크기 최적화

### Flutter
```dart
// 필요한 아이콘만 import
import 'package:refineui_system_icons/refineui_system_icons.dart';

// const 사용으로 성능 향상
const Icon(RefineUIIcons.add_24_filled)
```

### React
```jsx
// Tree shaking을 위한 개별 import
import { Add24Filled } from '@refineui/react-icons';

// 전체 라이브러리 import 피하기
// import * as Icons from '@refineui/react-icons';
```

### Web
```html
<!-- 필요한 CSS만 로드 -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/@refineui/web-icons@0.1.22/dist/refineui-system-icons.css">
```

## 메모리 사용량 최적화

### Flutter
```dart
class MyWidget extends StatelessWidget {
  // 아이콘을 static const로 캐싱
  static const IconData addIcon = RefineUIIcons.add_24_filled;
  
  @override
  Widget build(BuildContext context) {
    return Icon(addIcon);
  }
}
```

### React
```jsx
import { memo } from 'react';
import { Add24Filled } from '@refineui/react-icons';

// 자주 사용하는 아이콘을 memoize
const MemoizedIcon = memo(Add24Filled);
```

## 렌더링 성능 최적화

### Flutter
```dart
// RepaintBoundary로 아이콘 렌더링 최적화
RepaintBoundary(
  child: Icon(RefineUIIcons.add_24_filled),
)
```

### React
```jsx
// React.memo로 불필요한 리렌더링 방지
const IconButton = memo(({ icon, onClick }) => (
  <button onClick={onClick}>
    {icon}
  </button>
));
```
