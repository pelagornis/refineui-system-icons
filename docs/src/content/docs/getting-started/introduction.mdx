# Introduction to RefineUI System Icons

RefineUI System Icons is a cross-platform icon library for modern applications.

## What is it?

RefineUI System Icons is an icon library with the following features:

- **Cross-platform**: Supports Flutter, React, React Native, iOS, Android, Web
- **Consistent Design**: Provides the same visual experience across all platforms
- **Multiple Sizes**: Available in sizes from 16px to 48px
- **Two Styles**: Regular and Filled versions
