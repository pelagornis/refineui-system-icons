# 아이콘 검색

RefineUI System Icons에서 원하는 아이콘을 쉽게 찾을 수 있습니다.

## 검색 방법

### 이름으로 검색
아이콘의 이름을 입력하여 검색할 수 있습니다.

예시:
- `add` - 추가 관련 아이콘
- `home` - 홈 관련 아이콘
- `settings` - 설정 관련 아이콘

### 카테고리별 검색
아이콘을 카테고리별로 분류하여 찾을 수 있습니다.

- **기본**: add, remove, edit, delete
- **네비게이션**: home, back, forward, menu
- **액션**: play, pause, stop, refresh
- **상태**: check, error, warning, info
- **UI**: settings, user, notification, help

## 아이콘 미리보기

각 아이콘의 다양한 크기와 스타일을 미리 볼 수 있습니다:

- 16px Regular
- 16px Filled
- 20px Regular
- 20px Filled
- 24px Regular
- 24px Filled
- 28px Regular
- 28px Filled
- 32px Regular
- 32px Filled
- 48px Regular
- 48px Filled

## 코드 복사

각 아이콘의 사용법을 복사할 수 있습니다:

### Flutter
```dart
Icon(RefineUIIcons.add_24_filled)
```

### React
```jsx
<Add24Filled />
```

### Web
```html
<i class="refineui-icon refineui-icon-add-24-filled"></i>
```
