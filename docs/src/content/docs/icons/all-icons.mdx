# 모든 아이콘

RefineUI System Icons에서 제공하는 모든 아이콘을 확인할 수 있습니다.

## 아이콘 카테고리

### 기본 아이콘
- add, remove, edit, delete
- save, cancel, confirm
- search, filter, sort

### 네비게이션 아이콘
- home, back, forward
- menu, close, expand
- chevron-up, chevron-down, chevron-left, chevron-right

### 액션 아이콘
- play, pause, stop
- refresh, reload
- download, upload
- share, copy, paste

### 상태 아이콘
- check, error, warning, info
- success, failure
- loading, spinner

### 사용자 인터페이스
- settings, preferences
- user, profile, account
- notification, bell
- help, question

## 아이콘 크기

각 아이콘은 다음 크기로 제공됩니다:
- 16px
- 20px  
- 24px
- 28px
- 32px
- 48px

## 아이콘 스타일

각 아이콘은 두 가지 스타일로 제공됩니다:
- **Regular**: 기본 스타일
- **Filled**: 채워진 스타일

## 사용 예제

### Flutter
```dart
Icon(RefineUIIcons.add_24_filled)
Icon(RefineUIIcons.add_24_regular)
```

### React
```jsx
<Add24Filled />
<Add24Regular />
```

### Web
```html
<i class="refineui-icon refineui-icon-add-24-filled"></i>
<i class="refineui-icon refineui-icon-add-24-regular"></i>
```
