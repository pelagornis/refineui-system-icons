# 테마 설정

RefineUI System Icons의 테마 시스템을 사용하여 일관된 디자인을 적용하는 방법을 안내합니다.

## 다크 모드 지원

### Flutter
```dart
Icon(
  RefineUIIcons.add_24_filled,
  color: Theme.of(context).iconTheme.color,
)
```

### React
```jsx
const isDark = useTheme().isDark;

<Add24Filled 
  style={{ 
    color: isDark ? '#ffffff' : '#000000' 
  }} 
/>
```

### Web
```css
@media (prefers-color-scheme: dark) {
  .refineui-icon {
    color: #ffffff;
  }
}

.refineui-icon {
  color: #000000;
}
```

## 브랜드 색상 적용

### Flutter
```dart
class BrandColors {
  static const Color primary = Color(0xFF0078D4);
  static const Color secondary = Color(0xFF106EBE);
}

Icon(
  RefineUIIcons.add_24_filled,
  color: BrandColors.primary,
)
```

### React
```jsx
const brandColors = {
  primary: '#0078D4',
  secondary: '#106EBE',
};

<Add24Filled style={{ color: brandColors.primary }} />
```
