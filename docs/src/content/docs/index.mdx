## üéØ Quick Start

### Flutter

```dart
import 'package:refineui_system_icons/refineui_system_icons.dart';

Icon(RefineUIIcons.add_24_filled)
```

### React

```jsx
import { Add24Filled } from "@refineui/react-icons";

<Add24Filled />;
```

### Web

```html
<link rel="stylesheet" href="refineui-system-icons.css" />
<i class="refineui-icon refineui-icon-add-24-filled"></i>
```
