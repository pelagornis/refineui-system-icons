📦 CDN 사용법

CSS 로드

<link rel="stylesheet" href="https://jihoonahn.github.io/refineui-system-icons/cdn/refineui-system-icons-regular.css"> <link rel="stylesheet" href="https://jihoonahn.github.io/refineui-system-icons/cdn/refineui-system-icons-filled.css">

JavaScript 로드

<script src="https://jihoonahn.github.io/refineui-system-icons/cdn/index.umd.js"></script>

🎯 아이콘 사용 예시

CSS 클래스 방식

<span class="ic_refineui_gavel_24_regular"></span> <span class="ic_refineui_gavel_24_filled"></span>

JavaScript 함수 방식

const gavelIcon = getIconHTML('Gavel', 'regular', 24, 'red'); const gavel = Gavel(24, 'green'); const gavelFilled = GavelFilled(32, 'blue');

📁 사용 가능한 파일들

  • refineui-system-icons-regular.css - Regular 스타일 CSS
  • refineui-system-icons-filled.css - Filled 스타일 CSS
  • refineui-system-icons-regular.woff2 - Regular 폰트
  • refineui-system-icons-filled.woff2 - Filled 폰트
  • index.umd.js - JavaScript 라이브러리
